# E. HONDAFY - ストリートファイター エドモンド本田 どすこい変換器

『ストリートファイター』シリーズの力士キャラクター**「エドモンド本田 (E. HONDA)」**をモチーフにした超次元テキストコンバーターです。

入力された言葉の音節（モーラ）数、感嘆符・疑問符、語調をリアルタイムで解析し、ご要望の `ホンダホンダホダ` `ﾎﾝﾀﾞ` `ﾎﾝﾀﾞ!?` `ﾎｰﾝﾀﾞ"!!` に加え、**「どすこい！」「百裂張り手（フンフンフン…！）」「スーパー頭突き」「ごっつぁんです！」** などの熱いエドモンド本田語に変換します。

---

## 🌟 主な機能とモチーフ要素

1. **エドモンド本田 必殺技モード**
   - 💥 **半角カオス本田 (Chaos Honda)**: 原案の「`ホンダホンダホダ` `ﾎﾝﾀﾞ` `ﾎﾝﾀﾞ!?` `ﾎｰﾝﾀﾞ"!!`」を極めた超カオスモード。
   - ✋ **百裂張り手 (Hyakuretsu Harite)**: 「フンフンフンフン！張り手ぇー!!」と音節の分だけ張り手を高速連打！
   - 🐂 **スーパー頭突き (Super Zutsuki)**: 「どすこーーーーい!! ﾎｰﾝﾀﾞ"!!」と強烈な長音突進。
   - 🍲 **ごっつぁんちゃんこ (Gottsuan Style)**: 「ごっつぁんです！」「はっけよい！」「残った！」を交えた風格ある相撲口調。
   - 🏆 **世界一の相撲 (World Champion)**: 「わははは！相撲こそ世界一！ DOSUKOI!!」

2. **格ゲー風 HIT COUNTER (百裂ヒットカウンター)**
   - 文字数・音節数・テンションに連動して `100 HITS!` などのコンボヒッターを表示。

3. **マルチメディア演出**
   - 🔊 **Web Speech API どすこい朗読**: 低音で威勢の良い本田ボイスで変換結果を朗読。
   - 🎵 **Web Audio API 相撲SE**: 和太鼓・拍子木・百裂張り手連打音・スーパー頭突きインパクト音を内蔵。
   - ✨ **相撲スパーク演出**: 変換実行時に力士文字が飛散するパーティクルエフェクト。
