# CiNii 所蔵館チェッカー (CiNii Holdings Checker)

書誌ID (NCID) をもとに、CiNii Booksの所蔵情報を一括で確認し、他館所蔵の有無や所蔵館数、具体的な所蔵館名およびそのOPACリンクを調査するデスクトップアプリケーション（Windows用 executable）です。

## 主な機能
1. **書誌IDの一括検索**
   - **テキスト入力**: 画面に直接NCIDを改行区切りで貼り付けて検索可能。
   - **CSVファイル読み込み**: CSVファイルをドラッグ＆ドロップし、NCIDが記載されている列を指定して一括で読み込み可能。
2. **リアルタイムな進捗表示**
   - 処理の進捗率（％）やプログレスバー、成功・該当なし・エラーなどの統計情報をリアルタイムに確認可能。
   - CiNii APIへの負荷を抑えるためのリクエスト間隔（ウェイト）の調整機能や、処理の一時停止機能付き。
3. **インタラクティブな結果表示**
   - 検索結果の一覧テーブル（結果内検索・ステータスによる絞り込みが可能）。
   - 所蔵館数にマウスカーソルを当てると、**ツールチップで上位3館**を簡易表示。
   - 「詳細」ボタンまたは所蔵館数をクリックすると、書籍情報（タイトル、著者、出版社、出版年）と、**すべての所蔵館名およびその図書館OPACへのダイレクトリンク**をモーダル画面で一覧表示。
   - CiNii Booksの詳細ページへ直接アクセスするリンクも用意。
4. **確実なローカル動作・プロキシ対応**
   - **404 / CORS 回避**: 起動時に自動でローカルHTTPサーバーを背後で立ち上げ、アセットを安全にサーブするため、リソース読み込みエラーが起きません。
   - **プロキシ自動検出**: 通信に `requests` ライブラリを採用しており、OSのシステムプロキシ設定（社内・学内LAN等）を自動で読み込んで通信します。
5. **結果のエクスポート**
   - 検索結果を文字化けしないBOM付きUTF-8形式のCSVファイルとしてエクスポート可能。

## 使い方 (ユーザー向け)

### 1. アプリの起動
ビルド後に生成される `dist/CiNiiChecker.exe` をダブルクリックして起動します。

### 2. 検索方法
- **テキスト貼り付けの場合**:
  1. 「テキスト貼り付け」タブを選択します。
  2. テキストエリアに書誌ID（例: `BD18195266`、`BA89617462`）を改行で区切って入力します。
- **CSVファイルの場合**:
  1. 「CSVファイル」タブを選択します。
  2. CSVファイルをドラッグ＆ドロップするか、「ファイルを選択」から読み込ませます。
  3. 書誌IDが記載されている列インデックス（A列なら `0`、B列なら `1` ...）を設定します。

### 3. 実行
- 「リクエスト間隔」を設定します（デフォルトは `0.2` 秒）。
- **「検索開始」**ボタンをクリックします。
- 途中で停止したい場合は、**「一時停止」**をクリックします。

### 4. 結果の確認と詳細表示
- テーブルの「所蔵館数」の数字（例: `64 館`）にマウスカーソルを当てると、所蔵している上位3館がツールチップでポップアップ表示されます。
- テーブル右端の**「詳細」**ボタン、または「所蔵館数」の数字をクリックすると、詳細モーダルが開き、**所蔵館すべての名前と各館のOPACリンク**が表示されます。

### 5. CSVエクスポート
- 検索完了後、**「CSV出力」**ボタンをクリックすると、検索結果をCSVファイルとして保存できます。

## 開発者向け: ビルド方法（exe化）

Windows環境で `.exe` ファイルを作成するには、以下の手順を実行してください。

1. Python 3.8 以上がインストールされていることを確認します。
2. `build.bat` をダブルクリックして実行します。
   - 必要な依存ライブラリが自動でインストールされ、ビルドが始まります。
3. ビルドが正常に完了すると、同じディレクトリに `dist/` フォルダが作成され、その中に `CiNiiChecker.exe` が生成されます。

*※ビルド時に `pyinstaller` がうまく動かない場合は、PowerShellなどで以下のコマンドを直接実行してください。*
```bash
pip install -r requirements.txt pyinstaller
python -m PyInstaller --noconsole --onefile --add-data "ui;ui" --name "CiNiiChecker" app.py
```
