# Flowchart Studio (フローチャート構築ツール) 仕様・取扱説明書

本ドキュメントは、WEBブラウザ上でマウスクリック＆ドラッグ操作で自由に美しいフローチャートを作成できる「Flowchart Studio」の仕様、操作方法、技術仕様について記載したものです。

---

## 1. アプリケーション概要

Flowchart Studio は、外部ライブラリに依存せず、純粋な HTML5 / SVG / CSS3 / JavaScript（Vanilla JS）のみで構築されたグラフィカルなフローチャート作成ツールです。

直感的なマウス操作（ドラッグ＆ドロップ、端点接続、範囲選択、キャンバスパン＆ズーム）と、豊富なグラフィックカスタマイズ機能を備えています。

---

## 2. 主な機能と特徴

### 🎨 マウス＆タッチ操作による自由な構築
* **図形の配置**:
  * 左側の図形ライブラリからキャンバスへ「ドラッグ＆ドロップ」または「クリック」で図形を追加できます。
* **マウスでの直線・曲線接続**:
  * 各図形の端にある「青い円形ポート（端点）」から、別の図形のポートへドラッグするだけで、瞬時に矢印付きの接続線（エッジ）を引くことができます。
* **図形の位置移動 & サイズ調整**:
  * 図形をドラッグして自由移動。複数選択時は一括移動に対応。
  * 選択中図形の右下ハンドルをドラッグして伸縮リサイズ。
* **テキスト編集**:
  * 図形または接続線をダブルクリックすることで、その場でテキストやラベル（例：「はい」「いいえ」）を編集可能。

### 🛠️ 豊富な図形ライブラリ
* **開始・終了 (Start/End)**: ターミナル用カプセル形状
* **処理 (Process)**: 標準長方形
* **条件分岐 (Decision)**: ひし形（Yes/No分岐用）
* **データ入出力 (Input/Output)**: 平行四辺形
* **書類 (Document)**: 波底ドキュメント形状
* **データベース (Database)**: 円筒DB形状
* **サブプロセス (Subprocess)**: 二重枠線長方形
* **メモ (Note)**: 折り返し付メモ用紙
* **結合点 (Connector)**: 小円コネクタ

### 📐 キャンバス操作 & ガイド機能
* **スマートスナップ & グリッド**:
  * グリッド表示の切り替え（Snap to Grid）。
  * 20px間隔のグリッドに合わせた正確な位置揃え。
* **パン & ズーム (Pan & Zoom)**:
  * マウスホイール（Ctrl+ホイール）での拡縮（25%〜300%）。
  * 中クリックドラッグ、[Space]+ドラッグ、または「手のひらツール」での自由パン操作。
  * 「100%リセット」および「全図形フィット表示」ボタン搭載。
* **ラバーバンド（marquee）範囲選択**:
  * キャンバスの空白部分をドラッグして枠囲み選択。
  * 複数ノードの一括移動、左揃え・中央揃え・上揃えなどの整列処理、一括削除に対応。

### 💾 保存・書き出し (Export / Import)
* **PNG画像出力**: キャンバス上の高解像度PNG画像をダウンロード。
* **SVGベクトル画像出力**: 拡大しても劣化しないSVG形式で保存。
* **JSONプロジェクトファイル保存・読み込み**: 途中の作業状態を保存・いつでも復元可能。
* **Undo / Redo (元に戻す・やり直し)**: 最大40ステップの操作履歴を保持。
* **ダーク / ライト テーマ切替**: 目に優しいダークモードとクリーンなライトモードをワンタップ切替。

---

## 3. 動作サンプル (テンプレート)
初回起動時および「テンプレート」ボタンから、実用的なフローチャートのサンプルを即座に読み込んで試すことができます。

1. **ユーザーログイン認証フロー**: 条件分岐とDB照会、エラーリトライを含む基本認証処理。
2. **EC注文・在庫判定処理**: カート情報から在庫判定、決済、出荷命令データ連携に至る業務フロー。
3. **素数判定アルゴリズム**: 繰り返し計算（ループ）と条件分岐を用いたアルゴリズム表現。

---

## 4. キーボードショートカット一覧

| ショートカット | 機能 |
| :--- | :--- |
| `Delete` / `Backspace` | 選択中の図形・接続線を削除 |
| `Ctrl + Z` | 操作を元に戻す (Undo) |
| `Ctrl + Y` / `Ctrl + Shift + Z` | 操作をやり直す (Redo) |
| `Ctrl + D` | 選択中の図形を複製 (Duplicate) |
| `Ctrl + A` | すべての図形を選択 |
| `Space` + ドラッグ | キャンバスのパン移動 |
| `V` | 選択・移動ツール |
| `H` | 手のひら（パン）ツール |

---

## 5. 技術仕様
* **フロントエンド**: HTML5, SVG (Scalable Vector Graphics), CSS3, Vanilla JavaScript (ES6+)
* **ライブラリ依存**: なし（ビルドツール・サードパーティJS不要で完全スタンドアロン動作）
* **ブラウザ対応**: Google Chrome, Microsoft Edge, Safari, Firefox
