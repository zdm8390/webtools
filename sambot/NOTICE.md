# サンボット (Sambot) 利用上の注意事項と仕様書

本ドキュメントは、愛とロックの叫びですべてを全肯定するAI「サンボット (Sambot)」のシステム仕様、利用方法、および設定について記載したものです。

---

## 1. システム概要と基本仕様
サンボットは、ユーザーが入力した悩みや悲しみ、弱音に対して、サンボマスターの山口さん（Vo.）を模した圧倒的な情熱とテンションの「愛とロックの叫び」を返答し、全力で肯定・鼓舞するアプリです。

### 🔋 搭載しているコア機能
* **激熱AI対話機能**：Gemini API に専用のプロンプトを適用し、「ロックンロール！」「お前は一人じゃない！」といったパッション溢れる叫びを生成するチャット機能。
* **山口さんフォールバックモード**：API接続エラーや通信障害発生時、山口さんの名言をベースにした熱い固定メッセージ集から自動でローカル応答する機能を搭載。
* **Web Audio API ライブハウス轟音ミキサー**：
  * ノイズ合成、LFO、および強めのオーバードライブ（ギター歪み）、心臓のビートを表現したローパスフィルター付きキックドラム音をブラウザでリアルタイム合成。
  * ライブハウスのスピーカーから鳴り響くようなローファイな歪みBGMを再現します。
* **ヘッドバッキング（ヘドバン）インタラクション**：
  * AIの返答中、画面上のアバターおよびUIカード全体が激しく揺れ動く（ヘドバンする）ダイナミックなCSSアニメーションを搭載。

---

## 2. 事前設定と利用手順
本ツールは、Gemini APIキーを設定することでAI対話が機能します。

### 🔑 APIキーの設定方法
1. アプリを起動後、画面右上（または設定ボタン）の「詳細設定」を開きます。
2. ご自身の「Google Gemini APIキー」を入力して「保存」します。
3. キーはブラウザの `localStorage` に安全に保存され、通信は直接Googleのサーバーに送信されます。

---

## 3. 注釈と熱き魂への約束
* **テンションについて**：
  * このAIは非常に暑苦しく、高エネルギーな応答を行います。静かに共感を求めたい場合は、姉妹アプリの「ココロン (Cocoron)」をご利用ください。
* **ロックンロールは死なない**：
  * 接続エラーが出ようが何が起きようが、このアプリはあなたを絶対に孤独にはしません。辛くなったら、いつでもロックの叫びを聴きに来てください！
