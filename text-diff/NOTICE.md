# Text Diff Studio (コピペテキスト差分比較ツール) 仕様・取扱説明書

## 概要
Text Diff Studio は、ブラウザ上で2つのテキスト（ソースコード、文章、設定ファイルなど）をコピペして「⚡ 比較実行」を押すだけで、即座に行単位・単語単位の差分（追加・削除・変更箇所）を視覚的にハイライト表示する静的Webアプリケーションです。

---

## 🛠️ 主な機能と特徴

1. **コピペで即時比較**:
   - 左右のテキストエリアにテキストを張り付けるだけで、簡単に差分比較を実行できます。
2. **比較モードの切り替え**:
   - **行単位 (Line Diff)**: プログラムコードや文章の行追加・行削除をはっきりと可視化。
   - **単語・文字単位 (Word Diff)**: 文中の部分的な変更箇所やスペル修正をインラインでハイライト表示。
3. **柔軟な比較オプション**:
   - **前後の空白を無視**: インデントや改行前後のスペース違いを無視して差分抽出。
   - **大文字・小文字を無視**: アルファベットのケース違いを無視して比較。
4. **統計サマリー機能**:
   - 追加行数（緑 `+`）、削除行数（赤 `-`）、一致行数をリアルタイム集計して表示。
5. **スタンドアロン動作**:
   - 外部サーバーへのテキスト送信やAPI利用は一切行わず、すべての差分比較計算をブラウザローカル環境で高速に処理します。

---

## 🚀 利用方法

1. **テキストの準備**:
   - 左側のエリア `👈 変更前 (Original / Before)` に比較元のテキストを貼り付けます。
   - 右側のエリア `👉 変更後 (Modified / After)` に比較先のテキストを貼り付けます。
2. **比較の実行**:
   - ヘッダー右上の `⚡ 比較実行 (Compare)` ボタンを押すと、下の「📊 比較結果」に差分がレンダリングされます。
3. **サンプル読み込み / クリア**:
   - `📝 サンプル読み込み` ボタンで即座に動作テスト用コードが入力されます。
   - `🗑️ 入力をクリア` でテキストエリアを初期化できます。
