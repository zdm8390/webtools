# ココロン (Cocoron) 利用上の注意事項と仕様書

本ドキュメントは、全肯定型AIフレンド「ココロン (Cocoron)」のシステム仕様、利用方法、および設定について記載したものです。

---

## 1. システム概要と基本仕様
ココロンは、現代社会のストレスに寄り添うメンタルサポートAIです。ユーザーの悩みを否定せず、すべてを優しく全肯定し励ますメッセージを提供します。

### 🔋 搭載しているコア機能
* **全肯定AI対話機能**：Gemini API を利用した、徹底的にユーザーを肯定・共感するチャットボット。
* **ローカルフォールバックモード**：ネットワーク遮断やAPIエラー（過負荷等）が発生した際、AIの代わりに事前に内蔵された「優しい全肯定の言葉リスト」から即座に自動応答する仕組みを搭載。
* **Web Audio API ヒーリングシンセサイザー**：
  * 自然界の音（雨音、風音、焚き火のパチパチ音）を、ブラウザ上で**リアルタイムに波形合成（ホワイトノイズ・ブラウンノイズ・LFOフィルター）**してBGMとして再生します。
  * 音源ファイル（mp3等）をダウンロード・再生する形式ではないため、データ通信量がゼロで動作します。

---

## 2. 事前設定と利用手順
本ツールは、Gemini APIキーを設定することでAI対話が機能します。

### 🔑 APIキーの設定方法
1. アプリを起動後、画面右上（または設定ボタン）の「詳細設定」を開きます。
2. ご自身の「Google Gemini APIキー」を入力して「保存」します。
3. キーはブラウザの `localStorage` に安全に保存され、通信は直接Googleのサーバーに送信されます。

---

## 3. 注釈と技術的制限
* **AIの回答について**：
  * ココロンの応答はGeminiモデルによって生成されます。カウンセリングや医療行為の代替にはなりません。精神的苦痛が非常に強い場合は、適切な医療機関にご相談ください。
* **通信仕様**：
  * すべての対話処理は、入力したAPIキーを用いてユーザーのブラウザから直接GoogleのAPIエンドポイントにアクセスするため、当開発者に会話内容が送信・蓄積されることはありません。
