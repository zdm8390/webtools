# システム要件定義書および実装仕様書：『テトリスン』 (v3.0.0 Master Edition)

## 1. プロジェクト概要

* **タイトル**: テトリスン (Tetrisun)
* **バージョン**: `v3.0.0 Master Edition`
* **プラットフォーム**: Webブラウザ（GitHub Pagesによる静的ホスティング対応）
* **ターゲット動作環境**: PCおよびモバイル・タッチデバイス（レスポンシブ対応、アスペクト比 16:9 固定スケーリング）
* **操作方式**: キーボード（方向キー、A/S/Z/C/H/P、Space）、ゲームコントローラー（Gamepad API DAS自動連打追従）、およびモバイルタッチ画面コントローラー
* **世界観・デザイン方針**:
  * 魔法が存在する天空都市「パステリア王国」の動物たちによるテトリス対戦。
  * 白を基調としたパステルカラー、角が丸いPOPなテトリミノ、パステルオーロラグラデーション。
  * 爽快感を意識したエフェクト（ライン消去時シェイク、オノマトペ文字ポップ、カットイン、喜怒哀楽感情演出）。
  * 全画面がスクロールなしの1画面内に収まるレスポンシブUI構成。

## 2. 技術スタックおよびシステム構成

* **フロントエンド**: HTML5, CSS3 (Glassmorphism & Neon Glow), Pure JavaScript (ES6+)
* **描画ライブラリ**: HTML5 Canvas API（5 NEXT表示、HOLD表示、ゴースト破線、ハードドロップビーム線）
* **音響ライブラリ**: Web Audio API Synthesizer（全12種類のエフェクト音源・ピンチ時Danger BGMテンポ変調）
* **シナリオデータ形式**: YAML (`stories.yaml` 外部分離・感情タグ付き)
* **データ保存**: `localStorage`（ハイスコア、ランキング、設定情報、実績トロフィー情報）
* **オンライン対戦／乱入機能**: WebRTC (PeerJS) による P2P 接続（サーバーレス完結）

## 4. ゲームモード詳細

### 4.1 ストーリーモード
* **構成**: 難易度に応じた全ステージ進行（Easy: 3ステージ、Normal: 5ステージ、Hard: 7ステージ）。
* **会話劇システム**:
  * YAMLファイルから読み込んだ対話イベントを再生。
  * 発言者の丸型アバターアイコン（1P/敵カラー発光枠）を表示。
  * 喜怒哀楽感情バッジ（`😊` `🎉` `🔥` `😲` `✨` `🕶️` `💦`）と会話枠のダイナミック揺れ（`emotion-shake` / `bounce` / `pulse`）。

### 4.2 シングル対戦モード
* **形式**: CPUとの1対1対戦（AI思考エンジン搭載）。
* **乱入機能**: 乱入ONの場合、P2P接続を待受しオンライン対戦へ即時切替。

### 4.3 スコアアタックモード
* **形式**: シングルプレイのエンドレスモード。消去ライン数に伴いレベル・落下速度が自動上昇。

## 5. 高度パズルシステムおよび追加ルール仕様

### 5.1 競技テトリス互換仕様
* **5 NEXT キュー表示**: 次に出現するテトリミノを5個まで先読み表示。
* **SRS 3コーナー T-Spin 検出**: T型ピース回転時の3コーナー占有判定による T-Spin (Single/Double/Triple) 検出と専用ボーナススコア。
* **Perfect Clear (全消去) 判定**: 盤面が完全に空になった際に +2000pt ボーナスと虹色パーティクルファンファーレ演出。
* **リアルタイム PPS (Pieces Per Second) 計測**: 毎秒のブロック設置速度をリアルタイム計測・表示。
* **オノマトペ文字ポップ**: 消去時やハードドロップ時に「すとん♪」「きらーん★」「どかーん！」「ばばーん！！」の手描き風文字が弾ける演出。

### 5.2 魔法システムおよび効果
* **発動条件**: 経過時間 ＋ お邪魔被弾によるゲージ蓄積。ポーズ中はタイマーおよび発動を完全に停止。

  * **LV3**: 次に出現するテトリミノが5回連続で「I型（縦棒）」に固定化
  * **LV4**: キャラクター固有魔法
