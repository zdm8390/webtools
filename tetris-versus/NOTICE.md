# Cyber Tetris Versus (対戦型テトリス) 仕様・取扱説明書

本ドキュメントは、WEBブラウザ上で遊べるリアルタイム対戦型パズルゲーム「Cyber Tetris Versus」のシステム仕様、対戦ルール、オンラインマルチプレイヤー通信、キー操作、およびAIロジックについて記載したものです。

---

## 1. アプリケーション概要

Cyber Tetris Versus は、外部ゲームエンジンやサーバーインフラを必要とせず、純粋な **HTML5 Canvas / Web Audio API / WebRTC (PeerJS P2P) / CSS3 / Vanilla JavaScript** のみで構築されたリアルタイム対戦テトリスゲームです。

オンライン対戦（WebRTC P2P）、ローカル2人対戦（1P vs 2P）、高機能AI対戦（1P vs CPU）、および観戦モード（CPU vs CPU）に対応しています。

---

## 2. 🌐 オンライン対戦 (WebRTC P2P) システム

サーバーの会員登録や専用サーバー構築を必要とせず、ブラウザ同士が Peer-to-Peer (WebRTC) で直接通信してリアルタイム対戦を行います。

### 🔑 オンライン対戦の手順
1. **ルームを作成する (ホスト)**:
   * ヘッダーの **「🌐 オンライン対戦」** ボタンをクリック。
   * **「ルームコードを発行」** をクリックすると、一意のコード（例: `TETRIS-8A2F`）が生成されます。
   * 「コードをコピー」または **「招待リンクをコピー」** を押して対戦相手に共有します。
2. **ルームに参加する (ゲスト)**:
   * 共有されたコードを入力して「参加」ボタンを押すか、**招待URLを直接開く** と自動的にロビー経由で対戦が開始されます。

---

## 3. 搭載モード一覧

1. **🌐 オンライン対戦 (WebRTC P2P)**:
   - 遠隔地の友達・他端末のプレイヤーとネット越しに直接対戦。
2. **1P vs CPU (VS AIモード)**:
   - 人工知能CPUと対戦するモード。Easy (初級)、Medium (中級)、Hard (上級)、Master (鬼畜) の4段階の難易度。
3. **1P vs 2P (ローカル2人対戦モード)**:
   - 1台のPC・キーボードで2人で同時に操作して対戦。
4. **CPU vs CPU (観戦モード)**:
   - AI同士の高速対抗戦を観戦するモード。

---

## 4. 対戦・お邪魔ブロック（Garbage）システム

ライン消去に応じて、相手のフィールドの下部へ「お邪魔ブロック（Garbage Line）」を送り込むことができます。

### ⚡ 攻撃量テーブル
* **Single (1行消去)**: 0行
* **Double (2行消去)**: 1行
* **Triple (3行消去)**: 2行
* **Tetris (4行一括消去)**: 4行送出 💥
* **REN (コンボ)**: 連続でラインを消去するたびに追加攻撃ボーナス発生！

### 🛡️ お邪魔の相殺 (Counter / Offset) システム
* 相手から攻撃を受け取ると、ボード左側の**赤色警告ゲージ (Garbage Meter)** にスタックされます。
* 相手のお邪魔ブロックが自分の盤面にせり上がる前に自分もラインを消去すると、**受け取る予定のお邪魔ラインを直接打ち消して相殺（Counter）** できます。

## 5. 操作方法 (キーボード & タッチ)

### 🕹️ PLAYER 1 / オンライン自機
* **左 / 右 移動**: <kbd>A</kbd> / <kbd>D</kbd> または <kbd>←</kbd> / <kbd>→</kbd>
* **ソフトドロップ (高速落下)**: <kbd>S</kbd> または <kbd>↓</kbd>
* **ハードドロップ (即時接地)**: <kbd>W</kbd> または <kbd>↑</kbd>
* **回転 (右回転 / 左回転)**: <kbd>E</kbd> / <kbd>Q</kbd> (または <kbd>K</kbd> / <kbd>J</kbd>)
* **ホールド (キープ)**: <kbd>Space</kbd> または <kbd>Right Shift</kbd>

### 🎮 PLAYER 2 (ローカル2人対戦時)
* **左 / 右 移動**: <kbd>←</kbd> / <kbd>→</kbd>
* **ソフトドロップ**: <kbd>↓</kbd>
* **ハードドロップ**: <kbd>↑</kbd>
* **回転**: <kbd>K</kbd> / <kbd>J</kbd> (または <kbd>L</kbd>)
* **ホールド**: <kbd>Right Shift</kbd>

## 6. 技術仕様
* **P2P リアルタイム通信**: PeerJS (WebRTC DataConnection)
* **描画エンジン**: HTML5 2D Canvas (300x600 px 高解像度)
* **7-Bag / SRS**: 公式 Super Rotation System, Ghost Piece 着地表示
* **Web Audio API**: リアルタイム音源合成 (移動・回転・着地・消去・K.O.効果音)
