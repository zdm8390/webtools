# エレメンタル・フュージョン (Elemental Fusion) 仕様・取扱説明書

## 概要
『エレメンタル・フュージョン』は、落ち物パズルの連鎖感と、属性元素のシンカ（合体融合）システムを融合させたオリジナルファンタジーパズルゲームです。同じ属性の魔法宝珠を隣り合わせ、高位のエレメントへと成長させて大連鎖を目指します。

---

## 🎮 基本ルール・操作方法

### 1. 宝珠の操作
* 上部から落ちてくる2つの魔法エレメントペアを操作します。
* `<A>` / `<D>` または `←` / `→` : 左右移動
* `<W>` / `↑` / `Space` : 宝珠の回転
* `<S>` / `↓` : 高速落下 (ソフトドロップ)

### 2. 元素融合・連鎖システム
* 同じ属性（レベル）の宝珠が2つ以上隣り合うと、即座に融合（フュージョン）して1つ高位のエレメントへと進化します。
* 融合により生じた隙間に上部の宝珠が落下し、さらに連鎖して高位の宝珠へと進化すると `2 CHAIN!`, `3 CHAIN!` と連鎖が発動！
* 連鎖数が伸びるほど、爆発的なスコア倍率が得られます。
