# 10分サバイバーズ (10-Minute Survivors) 仕様・取扱説明書

## 概要
『10分サバイバーズ』は、画面を埋め尽くす敵の大群をなぎ倒しながら10分間の生き残りを目指す、迷宮ダンジョン脱出型2Dアクションゲームです。

---

## 🛠️ 主な機能と特徴

1. **🏰 迷宮ダンジョン & 脱出ポータル**:
   * プロシージャル（自動生成）された42x42セルで構成されるダンジョンマップ。
   * ミニマップと方位コンパスを頼りに、ダンジョン最奥に眠る **🚪 出口ポータル** を探して脱出勝利を目指します。
2. **⚔️ 3つのクラス選択**:
   * 🧙‍♂️ **Wizard (ウィザード)**: 魔法弾・範囲攻撃に優れたバランス型。
   * ⚔️ **Knight (ナイト)**: 高耐久・近接ブレードで無双する重装型。
   * 🏹 **Ranger (レンジャー)**: 高移動速度・クリティカル連射型。
3. **🛡️ バランス調整 & 無敵バリア**:
   * ダメージを受けると2.0秒間（無敵バリア）ダメージを無効化。
4. **🔊 動的サウンド合成**:
   * Web Audio API Synth による効果音。
