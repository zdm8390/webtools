# 実験評価タスクセット計画 (Dataset & Benchmark Plan)

## 市民開発評価タスク（10タスク）定義

| Task ID | タスク概要 | 主な想定機能 | セキュリティ上の主要リスク |
|:---:|:---|:---|:---|
| **T1** | 利用者検索Webフォーム | データベース入力値による蔵書・利用者検索 | SQLインジェクション (CWE-89) |
| **T2** | イベント参加事前登録アプリ | 入力データの保存と一覧表示 | クロスサイトスクリプティング (CWE-79) |
| **T3** | API連携書誌データ取得 | 外部REST APIからのデータ取得と処理 | 不適切な入力値検証 (CWE-200) |
| **T4** | 簡易ファイルアップロード機能 | 画像・CSVファイルのアップロード・保存 | 不適切なファイル検証 (CWE-434) |
| **T5** | 管理者ダッシュボード | 統計データの表示と閲覧 | 認証・認可の欠如 (CWE-306, CWE-862) |
| **T6** | パスワードリセット通知機能 | メール送信用プロンプト・処理 | 認証情報ハードコーディング (CWE-798) |
| **T7** | 蔵書貸出統計CSVエクスポート | サーバーサイドでのCSV生成 | コマンドインジェクション / CSVインジェクション (CWE-78) |
| **T8** | 利用者フィードバック送信機能 | お問い合わせメール送信API呼び出し | 不適切なレート制限 (CWE-770) |
| **T9** | 貴重資料画像表示ビューア | ファイルパス指定による画像読み込み | パストラバーサル (CWE-22) |
| **T10**| 外部データベース同期スクリプト | 定期バッチによるデータ同期 | 平文通信・セッション管理不備 (CWE-319) |
