# 文献検索戦略レポート (Search Strategy)

## 1. 段階的キーワード設計

- **Level 1 (テーマ)**: Citizen Development Security, LLM Code Generation Security
- **Level 2 (テーマ＋Method)**: LLM Code Generation + Multi-Agent System, Citizen Development + Governance Framework
- **Level 3 (テーマ＋Evaluation)**: GitHub Copilot Security Vulnerability Evaluation, LLM Generated Code Static Analysis
- **Level 4 (関連技術)**: OWASP Top 10 LLM, SAST / DAST Auto-remediation
- **Level 5 (同義語・関連語)**: Shadow IT, Low-Code / No-Code (LCNC), Automated Program Repair (APR)
- **Level 6 (略語)**: LLM, SAST, CWE, OWASP, DSR
- **Level 7 (日本語検索語)**: 市民開発 セキュリティ, AIコード生成 脆弱性, ガバナンス ガードレール

## 2. 検索データベース対象
- Tier 1: IEEE Xplore, ACM Digital Library (ソフトウェア工学・セキュリティ主要会議)
- Tier 2: Springer, Elsevier / ScienceDirect
- Tier 3: Google Scholar, J-STAGE, CiNii (国内研究・大学図書館関連)
- Tier 4: arXiv (最新LLMマルチエージェント動向)
